## Welcome

This site tracks the revival of the Pico Neo 2 headset. It's static, has no accounts, and all content is contributed via merge requests — no coding required.

## How to contribute

Check the [Contribute](#/contribute) page for instructions. Short version: edit a markdown file in `/data/`, open an MR, GitLab CI does the rest.

## What we track

- **Shims** — compatibility layers for running modern apps
- **Streaming** — PC and cloud streaming solutions
- **Decomp** — reverse engineering of the system and runtime
- **Projects** — community projects and research
- **Specs** — hardware and software reference docs
