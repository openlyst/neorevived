## Overview

The PicoXR Platform SDK is separate from the rendering/tracking SDK. It provides access to Pico platform services like user accounts, entitlement checks, leaderboards, achievements, and multiplayer features. This is the SDK you'd use if your app needs Pico Store integration or social features.

## Relationship to Other SDKs

- **PicoVR Unity SDK** handles rendering, tracking, and controller input
- **PicoXR Platform SDK** handles platform services (accounts, store, social)
- Both can be used together in the same Unity project
